# Reports from the public Bug Bounty Program

The swiyu Bug Bounty Program has been running since July 2025. It operates as part of the federal government’s Bug Bounty Program, which is overseen by the Federal Office for Cyber Security (BACS) and run in collaboration with Bug Bounty Switzerland AG.

Through such a program, ethical hackers are rewarded for identifying vulnerabilities. Until April 2026, this program has been run in private mode. This meant that only selected hackers could participate by invitation. 

From now on, the program will be run in public mode. This means that all hackers can be rewarded for vulnerabilities they find. The results of the swiyu Bug Bounty Program are published in this folder every quarter.
